eMule Shell Extension 1.1

Quick Start

1.      Install the shell extension by executing "eMule Shell Extension.msi"

2.      Use Windows Explorer to view contents of eMule's temporary directory.

Description

Windows Explorer GUI Integration

The eMule Shell Extension enables Windows Explorer to display additional information for eMule .part.met and .part files. The information is displayed in Tooltips, Statusbar, Detailpane and Detailview of Windows Explorer (see the attached screenshot).

 

However, it depends on your Windows Explorer configuration whether it is using any of the above mentioned GUI elements. Generally the information which can get displayed by Windows Explorer for eMule .part files is listed below:

 

Explorer Data

eMule Data

Title

Title of the file

Duration

Duration of an audio/video file

Data rate

Average data rate of an audio/video file

Artist

Artist (derived from MP3-ID3 information)

Album

Album (derived from MP3-ID3 information)

 

The information is displayed for .part.met files and for .part files which have an corresponding .part.met file located in the same directory.

Windows Shell Scripting Integration

The eMule Shell Extensions is registering itself as a file property provider for .part files in the Windows Shell. Therefore the existing Windows Shell Objects can be used to retrieve eMule .part file properties in the same way as for all other files.

 

Example

The following Windows Script is using standard shell objects to retrieve eMule .part file properties.

 

Set objShell = CreateObject("Shell.Application")

Set objFolder = objShell.Namespace("E:\eMule Temp")

 

Dim arrIDs(10)

arrIDs(0) = 0       ' Filename

arrIDs(1) = 1       ' Filesize

arrIDs(2) = 10     ' Title

arrIDs(3) = 21     ' Duration

arrIDs(4) = 22     ' Bitrate

arrIDs(5) = 16     ' Artist

arrIDs(6) = 17     ' Album

IDs = 7

 

' Get display names for properties

Dim arrHeaders(10)

For i = 0 to IDs - 1

    arrHeaders(i) = objFolder.GetDetailsOf(objFolder.Items, arrIDs(i))

Next

 

' Get property values for all "*.PART" files

Set folderItems = objFolder.Items

folderItems.Filter 64, "*.part"

For Each strFileName in folderItems

    For i = 0 to IDs - 1

       prop = objFolder.GetDetailsOf(strFileName, arrIDs(i))

       If prop <> "" Then

               Wscript.echo arrHeaders(i) & ": " & prop

        End If

    Next

    Wscript.echo

Next
